Interactive UR5e Kinematics Explorer

This interactive tool demonstrates the forward and inverse kinematics (FK/IK) of a UR5e robot arm. Adjust the sliders below to change the robot's joint angles. The 3D view will update to show the resulting configuration (red arm) and its end-effector (TCP) pose (large coordinate frame). The tool then calculates all possible inverse kinematics solutions that reach the same TCP pose, displaying them as semi-transparent arms.

How it Works

Forward Kinematics (FK): Given a set of joint angles (controlled by the sliders), FK calculates the precise 3D position and orientation of the robot's end-effector (Tool Center Point or TCP). The red robot arm and the large coordinate frame represent the FK result for the slider values.

Inverse Kinematics (IK): Given a desired TCP pose (position and orientation), IK calculates the necessary joint angles to achieve that pose. For a 6-axis robot like the UR5e, there can be multiple valid joint configurations (up to 8 for non-singular poses) that result in the same TCP pose. This tool calculates these solutions based on the pose derived from the sliders (the FK result) and visualizes them as the transparent arms.

The calculations are based on the Denavit-Hartenberg (DH) parameters of the UR5e and an analytical IK solver adapted from the Python code provided. Visualization is done using the three.js library.
